# GKGuard v0.1.10

## 中文

## 重点更新

- 发布依赖安全修复版本，确保新安装包使用更新后的 Electron 运行时。
- Electron 从 `33.4.11` 升级到 `42.4.0`。
- electron-builder 从 `25.1.8` 升级到 `26.15.3`。
- `tar` 依赖链升级到安全版本 `7.5.16`，解决 Dependabot 报告的高危路径遍历相关告警。
- Release workflow 的 Node.js 从 `20` 升级到 `22`，匹配 Electron 42 的 Node 版本要求。
- README 和 CONTRIBUTING 已补充本地桌面开发/打包需要 Node.js `22.12.0` 或更高版本。

## PR 与安全告警

- Dependabot PR #1 已由主分支更完整的依赖升级覆盖并自动关闭。
- GitHub Dependabot open alerts 已确认清零。
- 本地 `npm audit --audit-level=low` 已确认 0 vulnerabilities。

## 运行说明

- 下载 `GKGuard-Setup-0.1.10.exe` 安装后，打开即可启动内置本地 C2 后端并进入 `/demo`。
- C1 自动连接、内置服务器候选地址、SSH 密码提示、mock fallback 等 v0.1.9 功能保持不变。
- 已发布的 v0.1.9 安装包仍使用旧 Electron；需要安全依赖修复的用户应下载 v0.1.10。
- 本地开发或打包请使用 Node.js `22.12.0` 或更高版本。

## 验证情况

- `npm ci` 通过。
- `npm audit --audit-level=low`：0 vulnerabilities。
- Electron 主进程语法检查：`node --check desktop/main.js` 通过。
- 后端测试：`21 passed, 1 warning`。
- PyInstaller 后端包构建通过。
- 本地 Windows 安装包构建：`npm run dist` 通过。
- `git diff --check` 通过。

## 安装包

本 Release 下方包含：

- `GKGuard-Setup-0.1.10.exe`：Windows 桌面安装包。
- `GKGuard-Setup-0.1.10.exe.blockmap`：Electron 更新差分元数据。
- `latest.yml`：Electron 更新通道元数据。

普通安装只需要下载并运行 `GKGuard-Setup-0.1.10.exe`。

# GKGuard v0.1.10

## English

## Highlights

- Published a dependency security release so the new installer uses the updated Electron runtime.
- Upgraded Electron from `33.4.11` to `42.4.0`.
- Upgraded electron-builder from `25.1.8` to `26.15.3`.
- Upgraded the `tar` dependency chain to the safe `7.5.16` line, resolving high-severity path traversal alerts reported by Dependabot.
- Updated the Release workflow from Node.js `20` to Node.js `22` to match Electron 42 requirements.
- Updated README and CONTRIBUTING to state that local desktop development and packaging require Node.js `22.12.0` or later.

## PRs And Security Alerts

- Dependabot PR #1 was superseded by the more complete dependency upgrade on `main` and closed automatically.
- GitHub Dependabot open alerts were confirmed at 0.
- Local `npm audit --audit-level=low` was confirmed at 0 vulnerabilities.

## Runtime Notes

- Download and install `GKGuard-Setup-0.1.10.exe`, then open the app to start the bundled local C2 backend and load `/demo`.
- C1 auto-connection, built-in server candidates, SSH password prompt, and mock fallback from v0.1.9 remain unchanged.
- The already-published v0.1.9 installer still uses the old Electron runtime; users who need the dependency security fix should download v0.1.10.
- Use Node.js `22.12.0` or later for local development or packaging.

## Verification

- `npm ci` passed.
- `npm audit --audit-level=low`: 0 vulnerabilities.
- Electron main-process syntax: `node --check desktop/main.js` passed.
- Backend tests: `21 passed, 1 warning`.
- PyInstaller backend bundle build passed.
- Local Windows installer build: `npm run dist` passed.
- `git diff --check` passed.

## Installer Assets

This Release includes:

- `GKGuard-Setup-0.1.10.exe`: Windows desktop installer.
- `GKGuard-Setup-0.1.10.exe.blockmap`: Electron differential update metadata.
- `latest.yml`: Electron update-channel metadata.

For normal installation, download and run `GKGuard-Setup-0.1.10.exe`.
