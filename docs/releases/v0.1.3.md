# GKGuard v0.1.3

## 中文

## 重点更新

- 重建桌面 dashboard UI，让桌面端从基础演示页向更完整的工作台界面过渡。
- 保留 C2 mock 数据、报告、事件处置、审计和导出能力。
- 继续使用内置后端桌面包和标准化安装包命名。
- Release 资产保持 Windows 安装包、blockmap 和 `latest.yml` 三件套。

## 安装与网络说明

- 本版本可打开早期 C2 桌面演示界面。
- C2 mock 演示不依赖校园网；真实 C1、校园视频或服务器检索能力在该版本尚未接入。
- 当前展示和联调建议继续使用最新 Release。

## 验证情况

- GitHub Actions 成功构建 Windows 安装包。
- 已验证重建后的桌面 dashboard UI 可随 Release 发布流程使用。

## 安装包

本 Release 下方包含：

- `GKGuard-Setup-0.1.3.exe`：Windows 桌面安装包。
- `GKGuard-Setup-0.1.3.exe.blockmap`：Electron 更新差分元数据。
- `latest.yml`：Electron 更新通道元数据。

普通试用建议下载 `.exe` 文件；`.blockmap` 和 `latest.yml` 通常不需要手动打开。

# GKGuard v0.1.3

## English

## Highlights

- Rebuilt the desktop dashboard UI, moving the desktop app from a basic demo page toward a more complete operator workbench.
- Kept C2 mock data, reporting, event disposition, audit, and export capabilities.
- Continued using the bundled backend desktop package and standardized installer naming.
- Release assets remained the Windows installer, blockmap, and `latest.yml` trio.

## Installation And Network Notes

- This version can open the early C2 desktop demo UI.
- The C2 mock demo does not require campus network access; real C1, campus video, or server-side search was not integrated in this version.
- Use the latest Release for current presentations and integration work.

## Verification

- GitHub Actions successfully built the Windows installer.
- Verified that the rebuilt desktop dashboard UI could be delivered through the Release.

## Installer Assets

This Release includes:

- `GKGuard-Setup-0.1.3.exe`: Windows desktop installer.
- `GKGuard-Setup-0.1.3.exe.blockmap`: Electron differential update metadata.
- `latest.yml`: Electron update-channel metadata.

For normal testing, download the `.exe` file; `.blockmap` and `latest.yml` usually do not need to be opened manually.
