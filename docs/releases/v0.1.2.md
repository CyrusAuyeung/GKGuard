# GKGuard v0.1.2

## 中文

## 重点更新

- 将桌面发布版本推进到 `0.1.2`。
- 安装包命名切换为当前沿用的 `GKGuard-Setup-${version}.exe` 格式。
- 延续内置后端桌面包、GKGuard C2 mock dashboard、报告、审计和导出能力。
- 主要目标是修正发布版本号与资产命名，让 Release 页面更清晰。

## 安装与网络说明

- 该版本可用于早期 GKGuard C2 桌面演示，但仍建议当前演示使用最新 Release。
- GKGuard C2 mock 演示不依赖校园网；真实 CampusVision C1 服务、校园视频或服务器检索能力在该版本尚未接入。
- 从本版本开始，安装包文件名与 tag 版本号保持一致。

## 验证情况

- GitHub Actions 成功生成 `0.1.2` 版本 Windows 安装包。
- 已验证桌面安装资产、blockmap 和 `latest.yml` 可正常发布。

## 安装包

本 Release 下方包含：

- `GKGuard-Setup-0.1.2.exe`：Windows 桌面安装包。
- `GKGuard-Setup-0.1.2.exe.blockmap`：Electron 更新差分元数据。
- `latest.yml`：Electron 更新通道元数据。

普通试用建议下载 `.exe` 文件；`.blockmap` 和 `latest.yml` 通常不需要手动打开。

# GKGuard v0.1.2

## English

## Highlights

- Advanced the desktop release version to `0.1.2`.
- Switched installer naming to the current `GKGuard-Setup-${version}.exe` format.
- Continued the bundled backend desktop package, GKGuard C2 mock dashboard, report, audit, and export capabilities.
- This release mainly aligned release versioning and asset names so the Release page was clearer.

## Installation And Network Notes

- This version can be used for early GKGuard C2 desktop demos, but current demos should use the latest Release.
- The GKGuard C2 mock demo does not require campus network access; real CampusVision C1 service, campus video, or server-side search was not integrated in this version.
- Starting with this version, installer filenames match the tag version.

## Verification

- GitHub Actions successfully generated the `0.1.2` Windows installer.
- Verified that desktop installer assets, blockmap, and `latest.yml` could be published normally.

## Installer Assets

This Release includes:

- `GKGuard-Setup-0.1.2.exe`: Windows desktop installer.
- `GKGuard-Setup-0.1.2.exe.blockmap`: Electron differential update metadata.
- `latest.yml`: Electron update-channel metadata.

For normal testing, download the `.exe` file; `.blockmap` and `latest.yml` usually do not need to be opened manually.
