# GKGuard v0.1.19

## 更新重点

- 优化三屏人脸检索前端的响应式布局，最大化窗口会使用更多可用宽度，不再被固定窄容器限制。
- 将上传目标图、结果列表缩略图和详情关键帧改为完整显示，减少竖图、宽图和真实 C1 关键帧被裁切的问题。
- 调整小窗口布局和 Electron 桌面最小窗口尺寸，支持 `680x640` 下继续完成上传、结果查看、路线查看和重新上传流程。
- 在切换结果页和路线页时重置滚动位置，避免上一个页面的滚动状态影响下一步演示。
- 更新 GitHub Actions 发布工作流使用的 Action 运行时版本，降低后续发布链路因 Node.js 20 Action 运行时弃用提示受到影响的风险。
- 同步 README、API 规范、CampusVision C1 / GKGuard C2 集成说明、演示脚本、issue 模板和版本号到 `v0.1.19`。

## 验证

- 通过：`python -m pytest backend`，24 passed，1 个上游 deprecation warning。
- 通过：`node --check desktop/main.js` 和 `node --check desktop/preload.js`。
- 通过：`python -m compileall -q backend`。
- 通过：`npm audit --audit-level=low`，0 vulnerabilities。
- 通过：`git diff --check`。
- 通过：浏览器视觉检查 `1600x900`、`900x700`、`680x760` 和 `680x640`，确认无横向溢出、结果/路线页切换会重置滚动、控制台无新增报错。

## 发布文件

- `GKGuard-Setup-0.1.19.exe`：Windows 桌面安装包。
- `GKGuard-Setup-0.1.19.exe.blockmap`：Electron 更新差分元数据。
- `latest.yml`：Electron 自动更新元数据。

普通首次安装下载并运行 `GKGuard-Setup-0.1.19.exe`。

# GKGuard v0.1.19

## Highlights

- Improved the responsive layout for the three-screen face-search frontend so maximized windows use more available width instead of staying inside a narrow fixed container.
- Changed uploaded target images, result thumbnails, and detail keyframes to render fully, reducing cropping for portrait images, wide frames, and real C1 keyframes.
- Refined small-window layout behavior and the Electron desktop minimum window size so upload, result review, route review, and re-upload remain usable at `680x640`.
- Reset scroll position when switching between result and route screens so previous-page scroll state does not leak into the next demo step.
- Updated the GitHub Actions Release workflow action runtime versions, reducing the risk of future release instability from Node.js 20 action runtime deprecation warnings.
- Synchronized README, API specification, CampusVision C1 / GKGuard C2 integration notes, demo script, issue templates, and package versions to `v0.1.19`.

## Verification

- Passed: `python -m pytest backend`, 24 passed with 1 upstream deprecation warning.
- Passed: `node --check desktop/main.js` and `node --check desktop/preload.js`.
- Passed: `python -m compileall -q backend`.
- Passed: `npm audit --audit-level=low`, 0 vulnerabilities.
- Passed: `git diff --check`.
- Passed: browser visual checks at `1600x900`, `900x700`, `680x760`, and `680x640`, confirming no horizontal overflow, scroll reset between result/route screens, and no new console errors.

## Release Assets

- `GKGuard-Setup-0.1.19.exe`: Windows desktop installer.
- `GKGuard-Setup-0.1.19.exe.blockmap`: Electron differential update metadata.
- `latest.yml`: Electron auto-update metadata.

For a first-time installation, download and run `GKGuard-Setup-0.1.19.exe`.
