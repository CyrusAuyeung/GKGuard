# GKGuard v0.1.0

## 中文

## 重点更新

- 建立 GKGuard C2 后端演示基础，包括人员、事件、报告、审计和导出相关接口雏形。
- 加入可视化演示 dashboard，形成早期 GKGuard C2 操作台体验。
- 增加案件报告生成、事件处置归档、敏感操作审计和案件包导出能力。
- 对中文 dashboard 进行可用性和视觉打磨，并重建主要页面布局。
- 首次加入 Electron 桌面 Release workflow，可由 GitHub Actions 生成 Windows 安装资产。

## 安装与网络说明

- 这是最早的桌面发布版本，主要用于验证 GKGuard C2 壳体和自动发版链路。
- 建议当前演示或联调使用最新版本；v0.1.0 不包含后续版本里的完整内置后端和 CampusVision C1 服务接入能力。
- GKGuard C2 mock 演示不依赖校园网；真实 CampusVision C1 服务、校园视频或服务器检索能力在该版本尚未接入。

## 验证情况

- GitHub Actions 成功生成 Windows 安装包、blockmap 和 `latest.yml`。
- 已完成早期 GKGuard C2 mock API、报告、审计和 dashboard 演示链路验证。

## 安装包

本 Release 下方包含：

- `GKGuard-Setup-0.1.0.exe`：Windows 桌面安装包。
- `GKGuard-Setup-0.1.0.exe.blockmap`：Electron 更新差分元数据。
- `latest.yml`：Electron 更新通道元数据。

普通试用建议下载 `.exe` 文件；`.blockmap` 和 `latest.yml` 通常不需要手动打开。

# GKGuard v0.1.0

## English

## Highlights

- Established the initial GKGuard C2 backend demo foundation, including early people, event, report, audit, and export APIs.
- Added a visual demo dashboard to shape the first GKGuard C2 operator-console experience.
- Added case report generation, event disposition archive, sensitive-action audit logging, and case package export.
- Polished the Chinese dashboard for usability and rebuilt the main layout.
- Added the first Electron desktop Release workflow so GitHub Actions could produce Windows release assets.

## Installation And Network Notes

- This is the earliest desktop release and mainly validates the GKGuard C2 shell plus automated release flow.
- Use the latest version for current demos or integration work; v0.1.0 does not include the later fully bundled backend or CampusVision C1 service integration.
- The GKGuard C2 mock demo does not require campus network access; real CampusVision C1 service, campus video, or server-side search was not integrated in this version.

## Verification

- GitHub Actions successfully generated the Windows installer, blockmap, and `latest.yml`.
- Early GKGuard C2 mock APIs, reports, audit logs, and dashboard demo flow were validated.

## Installer Assets

This Release includes:

- `GKGuard-Setup-0.1.0.exe`: Windows desktop installer.
- `GKGuard-Setup-0.1.0.exe.blockmap`: Electron differential update metadata.
- `latest.yml`: Electron update-channel metadata.

For normal testing, download the `.exe` file; `.blockmap` and `latest.yml` usually do not need to be opened manually.
