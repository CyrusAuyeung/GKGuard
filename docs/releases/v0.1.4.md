# GKGuard v0.1.4

## 中文

## 重点更新

- 优化 Electron 桌面壳布局，让桌面窗口更接近正式工作台形态。
- 调整界面结构、窗口尺寸和主要内容区域，提升桌面端可读性。
- 继续保留 GKGuard C2 mock dashboard、报告、处置、审计和导出能力。
- 保持 GitHub Actions 自动构建 Windows 安装包。

## 安装与网络说明

- 本版本可打开 GKGuard C2 桌面壳和本地 mock 演示。
- GKGuard C2 mock 演示不依赖校园网；真实 CampusVision C1 服务、校园视频或服务器检索能力在该版本尚未接入。
- 当前展示和联调建议使用最新 Release。

## 验证情况

- GitHub Actions 成功构建 Windows 安装包。
- 已验证桌面壳布局调整可进入安装包构建流程。

## 安装包

本 Release 下方包含：

- `GKGuard-Setup-0.1.4.exe`：Windows 桌面安装包。
- `GKGuard-Setup-0.1.4.exe.blockmap`：Electron 更新差分元数据。
- `latest.yml`：Electron 更新通道元数据。

普通试用建议下载 `.exe` 文件；`.blockmap` 和 `latest.yml` 通常不需要手动打开。

# GKGuard v0.1.4

## English

## Highlights

- Refined the Electron desktop shell layout so the window felt closer to a production workbench.
- Adjusted page structure, window sizing, and main content areas to improve desktop readability.
- Kept GKGuard C2 mock dashboard, report, disposition, audit, and export capabilities.
- Continued automatic Windows installer builds through GitHub Actions.

## Installation And Network Notes

- This version can open the GKGuard C2 desktop shell and local mock demo.
- The GKGuard C2 mock demo does not require campus network access; real CampusVision C1 service, campus video, or server-side search was not integrated in this version.
- Use the latest Release for current presentations and integration work.

## Verification

- GitHub Actions successfully built the Windows installer.
- Verified that the refined desktop shell layout could be delivered in the installer.

## Installer Assets

This Release includes:

- `GKGuard-Setup-0.1.4.exe`: Windows desktop installer.
- `GKGuard-Setup-0.1.4.exe.blockmap`: Electron differential update metadata.
- `latest.yml`: Electron update-channel metadata.

For normal testing, download the `.exe` file; `.blockmap` and `latest.yml` usually do not need to be opened manually.
