# GKGuard v0.1.17

## 更新重点

- 重画上传、检查更新、路线图、导出和信息提示图标，去掉不自然的组合感，并保持按钮内图标与文字颜色一致。
- 修正信息圆圈中的提示符号居中问题，使首页提示条和路线摘要提示保持一致。
- 调整路线页工具栏顺序，将 `重新上传` 固定为最左侧按钮，后续依次为返回结果、导出路线图和查看完整轨迹。
- 保留上一版已接入的品牌图标资产、桌面应用图标、内嵌 SVG sprite 方案和应用内更新流程。

## 验证

- 通过：`python -m pytest backend`，24 passed，1 个上游 deprecation warning。
- 通过：`node --check desktop/main.js` 和 `node --check desktop/preload.js`。
- 通过：本地预览页面检查，确认 SVG symbol 存在、按钮图标无背景贴图、路线页按钮顺序正确。
- 通过：`npm audit --audit-level=low`，0 vulnerabilities。
- 通过：`npm run dist`，已生成 `GKGuard-Setup-0.1.17.exe` 和 `.blockmap`。

## Release 资产

- `GKGuard-Setup-0.1.17.exe`：Windows 桌面安装包。
- `GKGuard-Setup-0.1.17.exe.blockmap`：Electron 更新差分元数据。
- `latest.yml`：Electron 自动更新元数据。

普通首次安装下载并运行 `GKGuard-Setup-0.1.17.exe`。

# GKGuard v0.1.17

## Highlights

- Redrew the upload, update-check, route-map, export, and information icons to remove the unnatural assembled look while keeping button icons aligned with surrounding text color.
- Fixed the centered information mark inside the circular notice icon so the home banner and route summary notice read consistently.
- Reordered the route-screen toolbar so `重新上传` stays as the leftmost action, followed by back to results, export route map, and view full trajectory.
- Preserved the previous release's brand icon assets, desktop app icon, embedded SVG sprite strategy, and in-app update flow.

## Verification

- Passed: `python -m pytest backend`, 24 passed with 1 upstream deprecation warning.
- Passed: `node --check desktop/main.js` and `node --check desktop/preload.js`.
- Passed: local preview inspection for SVG symbol presence, no icon background images, and correct route toolbar order.
- Passed: `npm audit --audit-level=low`, 0 vulnerabilities.
- Passed: `npm run dist`, producing `GKGuard-Setup-0.1.17.exe` and `.blockmap`.

## Release Assets

- `GKGuard-Setup-0.1.17.exe`: Windows desktop installer.
- `GKGuard-Setup-0.1.17.exe.blockmap`: Electron differential update metadata.
- `latest.yml`: Electron auto-update metadata.

For a first-time installation, download and run `GKGuard-Setup-0.1.17.exe`.
