# GKGuard v0.1.32

## 重点更新

- 修复结果页左侧人物照片在方框内二次显示不完整的问题：人物照片现在按裁切图自身比例完整缩放进方框，不再被正方形图片元素拉伸或视觉截断。
- 保持上传区和放大选人弹窗的人脸框定位逻辑不变，避免影响查询图人脸框坐标对齐。
- 补充 E2E 回归：验证结果页人物照片的渲染比例接近图片自然比例，并且图片矩形完整落在人物照片方框内。
- 同步 README、API 规范、CampusVision C1 / GKGuard C2 集成说明、自动连接说明、演示脚本、文档索引、图标说明和版本号到 `v0.1.32`。

## 行为说明

- 人物照片裁切图本身仍来自用户选中的目标人脸框；本版只修复结果页方框内的显示方式，让裁切图完整展示。
- 本版本不修改远端 CampusVision C1 服务，不改变相似度计算和检索排序逻辑，也不向仓库加入真实监控、人脸或个人信息数据。

- 通过：`python -m pytest services/campusvision-c1/tests`，本地环境共 `2 skipped`。
- 通过：`npm run test:e2e`，桌面、紧凑和移动视口共 `18 passed`。
- 通过：`npm audit --audit-level=low`，`0 vulnerabilities`。
- 通过：`git diff --check`。
- 通过：`npm run dist`，本地生成 `GKGuard-Setup-0.1.32.exe` 和 `.blockmap`。
- 发布流程要求 GitHub Actions `Verify` 和 Release workflow 通过，并在正式 GitHub Release 中核对安装包、`.blockmap` 和 `latest.yml` 齐全。

## 安装

普通首次安装下载并运行 `GKGuard-Setup-0.1.32.exe`。已安装旧版时，可使用软件右上角 `检查更新` 入口更新。

真实 CampusVision C1 检索仍需要校园网、VPN 或 SSH 隧道能访问 CampusVision C1 服务；未上传图片时可使用本地模拟流程演示，上传图片后的检测失败、无匹配、超时或真实检索失败不会展示模拟命中。

# GKGuard v0.1.32

## Highlights

- Fixed result-side portrait images not fully displaying inside the portrait frame: portraits now scale into the frame using their own image ratio instead of being stretched into a square image element.
- Kept upload-preview and enlarged face-selection box positioning unchanged so query-face overlay alignment is not affected.
- Added an E2E regression that verifies the result portrait render ratio stays close to the image natural ratio and the rendered image rectangle remains fully inside the portrait frame.
- Synchronized README, API specification, CampusVision C1 / GKGuard C2 integration notes, auto-connection guide, demo script, documentation index, icon notes, and package versions to `v0.1.32`.

## Behavior Notes

- The portrait crop still comes from the user-selected target face box. This release only fixes how that crop is displayed inside the result-side portrait frame so the crop is fully visible.
- This release does not modify the remote CampusVision C1 service, does not change similarity scoring or result ranking, and does not add real surveillance, face, or personal-information data to the repository.

## Installation

For a first-time installation, download and run `GKGuard-Setup-0.1.32.exe`. Existing installations can update through the top-right `检查更新` entry.

Real CampusVision C1 search still requires campus network, VPN, or an SSH tunnel to the CampusVision C1 service. The mock flow remains available when no image has been uploaded; uploaded-image detection failures, no-match results, timeouts, or real-search failures do not show mock hits.
