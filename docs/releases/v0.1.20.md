# GKGuard v0.1.20

## 更新重点

- 修复桌面端可能复用旧 `127.0.0.1:8000` 后端的问题：启动时会检查 `/demo` 页面引用的静态资源版本，不匹配时自动切换到空闲端口启动当前后端，避免新版壳体加载旧页面资源。
- 进一步放宽结果页和路线页布局，让最大化窗口和宽屏窗口使用更多可用宽度，不再停留在中间窄区域。
- 保持结果页在 `680x640` 小窗口下无横向溢出，并验证宽屏 `2048x1086` 下结果页主内容可展开到约 1977px。
- 将 CampusVision C1 结果记录拆分为 `faceUrl` 和 `frameUrl`：记录列表优先显示 CampusVision C1 人脸裁剪缩略图，详情区显示完整关键帧。
- 将上传图、CampusVision C1 人脸裁剪缩略图和关键帧保持为 `object-fit: contain`，减少真实 CampusVision C1 宽图、竖图或监控关键帧被裁切的问题。
- 同步 README、API 规范、CampusVision C1 / GKGuard C2 集成说明、数据字典、演示脚本、支持/安全文档、issue 模板和版本号到 `v0.1.20`。

## 验证

- 通过：`python -m pytest backend`，25 passed，1 个上游 deprecation warning。
- 通过：`node --check desktop/main.js`。
- 通过：`python -m py_compile backend/desktop_server.py`。
- 通过：`git diff --check`，仅 Windows CRLF 提示。
- 通过：浏览器检查 `680x640` 和 `2048x1086` 视口，确认结果页无横向溢出、加载 `resultlayout3` 静态资源、宽屏布局展开、缩略图和关键帧使用 `object-fit: contain`。
- 通过：本地旧后台复用模拟，确认旧静态资源服务不会被新版桌面壳复用。

## 发布文件

- `GKGuard-Setup-0.1.20.exe`：Windows 桌面安装包。
- `GKGuard-Setup-0.1.20.exe.blockmap`：Electron 更新差分元数据。
- `latest.yml`：Electron 自动更新元数据。

普通首次安装下载并运行 `GKGuard-Setup-0.1.20.exe`。

# GKGuard v0.1.20

## Highlights

- Fixed a desktop startup issue where the app could reuse an old backend already listening on `127.0.0.1:8000`. The shell now checks the static asset version referenced by `/demo` and falls back to a free port when the running backend is stale.
- Further widened the result and route layouts so maximized and wide windows use more available space instead of remaining in a narrow centered area.
- Kept the result page free of horizontal overflow at `680x640`, and verified that the wide `2048x1086` viewport expands the result page to about 1977px.
- Split CampusVision C1 media fields into `faceUrl` and `frameUrl`: record lists prefer CampusVision C1 face-crop thumbnails, while the detail panel displays the complete keyframe.
- Kept uploaded images, CampusVision C1 face-crop thumbnails, and keyframes on `object-fit: contain`, reducing cropping for real CampusVision C1 portrait images, wide frames, and surveillance keyframes.
- Synchronized README, API specification, CampusVision C1 / GKGuard C2 integration notes, data dictionary, demo script, support/security docs, issue templates, and package versions to `v0.1.20`.

## Verification

- Passed: `python -m pytest backend`, 25 passed with 1 upstream deprecation warning.
- Passed: `node --check desktop/main.js`.
- Passed: `python -m py_compile backend/desktop_server.py`.
- Passed: `git diff --check`, with only Windows CRLF notices.
- Passed: browser checks at `680x640` and `2048x1086`, confirming no horizontal overflow, `resultlayout3` static assets, expanded wide layout, and `object-fit: contain` for thumbnails and keyframes.
- Passed: local stale-backend reuse simulation, confirming the newer desktop shell does not reuse an older static-resource server.

## Release Assets

- `GKGuard-Setup-0.1.20.exe`: Windows desktop installer.
- `GKGuard-Setup-0.1.20.exe.blockmap`: Electron differential update metadata.
- `latest.yml`: Electron auto-update metadata.

For a first-time installation, download and run `GKGuard-Setup-0.1.20.exe`.
