# GKGuard v0.1.16

## 更新重点

- 将桌面 `icons` 文件夹中的编号图标规范命名并接入仓库品牌资产目录。
- 前端标题、上传、检索、重新上传、返回结果、路线图、导出、检查更新和信息提示图标改为内嵌线性矢量图标，跟随按钮文字颜色自然融合。
- 桌面启动页保留品牌图标资产，CampusVision C1 服务器密码窗口改为线性连接图标。
- Windows 安装包和 Electron 窗口指定 GKGuard 应用图标，避免继续使用默认 Electron 图标。
- 保留 `v0.1.15` 的 CampusVision C1 连接进度、快速 tunnel 探测和无滚动密码窗口修复。

## 图标方案

- `app-mark.png` / `app-mark.ico`：应用主标识和安装包图标。
- `boot-mark.png`：启动页品牌图标。
- 功能小图标：使用页面内 SVG sprite 和 `currentColor`，避免 PNG 白底和贴图感。

## 验证

- 通过：`python -m pytest backend`，24 passed，1 个上游 deprecation warning。
- 通过：`node --check desktop/main.js` 和 `node --check desktop/preload.js`。
- 通过：`npm ci` 和 `npm audit --audit-level=low`，0 vulnerabilities。
- 通过：`npm run dist`，已生成 `GKGuard-Setup-0.1.16.exe` 和 `.blockmap`。

## Release 资产

- `GKGuard-Setup-0.1.16.exe`：Windows 桌面安装包。
- `GKGuard-Setup-0.1.16.exe.blockmap`：Electron 更新差分元数据。
- `latest.yml`：Electron 自动更新元数据。

普通首次安装下载并运行 `GKGuard-Setup-0.1.16.exe`。

# GKGuard v0.1.16

## Highlights

- Renamed and imported the numbered icons from the desktop `icons` folder into repository brand asset directories.
- Replaced headings, upload, search, re-upload, back-to-results, route map, export, update check, and information notice icons with embedded linear vector icons that naturally follow the surrounding text color.
- Kept the brand icon asset on the desktop loading page and changed the CampusVision C1 server password window to a linear connection icon.
- Configured the Windows installer and Electron windows to use the GKGuard app icon instead of the default Electron icon.
- Preserved the `v0.1.15` CampusVision C1 connection progress, fast tunnel probing, and non-scrolling password window fixes.

## Icon Strategy

- `app-mark.png` / `app-mark.ico`: app mark and installer icon.
- `boot-mark.png`: loading page brand icon.
- Functional UI icons: embedded SVG sprite with `currentColor`, avoiding PNG white backgrounds and pasted-image feel.

## Verification

- Passed: `python -m pytest backend`, 24 passed with 1 upstream deprecation warning.
- Passed: `node --check desktop/main.js` and `node --check desktop/preload.js`.
- Passed: `npm ci` and `npm audit --audit-level=low`, 0 vulnerabilities.
- Passed: `npm run dist`, producing `GKGuard-Setup-0.1.16.exe` and `.blockmap`.

## Release Assets

- `GKGuard-Setup-0.1.16.exe`: Windows desktop installer.
- `GKGuard-Setup-0.1.16.exe.blockmap`: Electron differential update metadata.
- `latest.yml`: Electron auto-update metadata.

For a first-time installation, download and run `GKGuard-Setup-0.1.16.exe`.
