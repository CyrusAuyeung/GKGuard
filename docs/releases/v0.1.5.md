# GKGuard v0.1.5

## 中文

## 重点更新

- 完善桌面壳交互，让主要按钮、筛选、导航和工作台状态更接近可演示版本。
- 补齐前端静态壳体中的核心演示路径，降低“只有界面不能操作”的问题。
- 继续保留 C2 本地 mock 数据，用于离线演示 AI 搜索、事件处置和审计流程。
- Windows 安装包继续由 GitHub Actions 构建并发布。

## 安装与网络说明

- 本版本可打开较完整的 C2 桌面壳交互演示。
- C2 mock 演示不依赖校园网；真实 C1、校园视频或服务器检索能力在该版本尚未接入。
- 如需当前 C1 真实检索，请使用 v0.1.8 或更新版本。

## 验证情况

- GitHub Actions 成功构建 Windows 安装包。
- 已验证桌面壳主要交互和本地 mock 演示路径可运行。

## 安装包

本 Release 下方包含：

- `GKGuard-Setup-0.1.5.exe`：Windows 桌面安装包。
- `GKGuard-Setup-0.1.5.exe.blockmap`：Electron 更新差分元数据。
- `latest.yml`：Electron 更新通道元数据。

普通试用建议下载 `.exe` 文件；`.blockmap` 和 `latest.yml` 通常不需要手动打开。

# GKGuard v0.1.5

## English

## Highlights

- Completed more desktop-shell interactions so main buttons, filters, navigation, and workbench states were closer to a demo-ready version.
- Filled in the core frontend shell paths to reduce the “static UI only” gap.
- Kept local C2 mock data for offline AI search, event disposition, and audit demonstrations.
- Windows installer builds continued through GitHub Actions.

## Installation And Network Notes

- This version can open a more complete C2 desktop-shell interaction demo.
- The C2 mock demo does not require campus network access; real C1, campus video, or server-side search was not integrated in this version.
- Use v0.1.8 or later for the current real C1 search integration.

## Verification

- GitHub Actions successfully built the Windows installer.
- Verified the main desktop shell interactions and local mock demo paths.

## Installer Assets

This Release includes:

- `GKGuard-Setup-0.1.5.exe`: Windows desktop installer.
- `GKGuard-Setup-0.1.5.exe.blockmap`: Electron differential update metadata.
- `latest.yml`: Electron update-channel metadata.

For normal testing, download the `.exe` file; `.blockmap` and `latest.yml` usually do not need to be opened manually.
